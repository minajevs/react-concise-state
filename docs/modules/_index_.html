createStoreContext

  • createStoreContext<TState, TContexts, TActions>(initialState: TState, actions?: TActions, contexts?: TContexts): [Context<Store<TState, TContexts, TActions>>, React.FC]
    • Defined in index.tsx:152

    Creates react context with state store. Store consists of state properties and actions which modify it.

    Type parameters

    • TState

      any JS object representing application or application part state

    • TContexts: Contexts

      a dictionary (key-value object) of injected store contexts

    • TActions: Actions<TState, TContexts>

      creator function of a dictionary (key-value object) of action creators

    Parameters

    • initialState: TState

      initial state

    • Default value actions: TActions = {} as TActions

      is a function which creates dictionary (key-value object) of action creators for a given store. Actions might have arguments, can modify state and return values. Actions creator provide special ContextReference object for all action-creators, which has:

      • setState method to modify state
      • state object to access current state of type
      • stores object to access injected store contexts

      Any argument in action creator will be converted to action parameter, which user must provide when calling action.

      Example:

      • Action creator:
        (reference) => {
   actionName: (payload: string) => {
        reference.setState({whatever: payload})
   }
}
      • Action:
        store.actionName("Hello world")
    • Default value contexts: TContexts = {} as TContexts

      a dictionary (key-value object) of injected store contexts. Those stores will be resolved and passed to actions in stores property of ContextReference

      Example:

      createStoreContext({foo: '123'}, (reference) => ({
 action: () => {
     reference.stores.bar.someBarAction()
     return reference.stores.bar.someBarProperty
 }
}), {
 bar: barContext
})

    Returns [Context<Store<TState, TContexts, TActions>>, React.FC]

    tuple, where context is [[React.Context]] and should be consumed to access store, and Provider is [[React.FC]] (functional component) without props, which enables store actions for any nested components
